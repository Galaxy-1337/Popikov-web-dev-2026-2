# Лабораторная работа №2 — Flask: запросы и формы

Веб-приложение на **Flask**, демонстрирующее работу с параметрами запроса, заголовками, cookies, формами и валидацией данных.

## 🛠 Стек

- **Backend:** Python 3, Flask 3.0
- **Шаблоны:** Jinja2
- **Стили:** Vanilla CSS (тёмная тема, Inter + Playfair Display) + Bootstrap 5 (только `is-invalid`/`invalid-feedback`)
- **Тесты:** pytest

## 🗂 Структура

```
Лабораторная 2/
├── requirements.txt
├── run.bat
└── app/
    ├── app.py           # Маршруты Flask + логика валидации телефона
    ├── __init__.py
    ├── .env
    ├── static/
    │   └── styles.css   # Все стили
    ├── templates/
    │   ├── base.html    # Базовый шаблон (navbar, footer)
    │   ├── index.html   # Главная (карточки разделов)
    │   ├── url_params.html   # Параметры URL
    │   ├── headers.html      # Заголовки запроса
    │   ├── cookies.html      # Cookie
    │   ├── form_params.html  # Параметры формы
    │   └── phone.html        # Валидация телефона
    └── tests/
        └── test_app.py  # 26 тест-кейсов
```

## 🚀 Запуск

### Вариант 1 — через bat-скрипт
```
run.bat
```

### Вариант 2 — вручную
```bash
pip install -r requirements.txt
python app/app.py
```

Приложение откроется на **http://127.0.0.1:5000**

## 📄 Маршруты

| URL              | Метод      | Описание                                      |
|------------------|------------|-----------------------------------------------|
| `/`              | GET        | Главная страница (карточки разделов)          |
| `/url-params`    | GET        | Параметры URL (`request.args`)                |
| `/headers`       | GET        | Заголовки запроса (`request.headers`)         |
| `/cookies`       | GET        | Cookie — устанавливает/удаляет куку           |
| `/form-params`   | GET, POST  | Параметры формы (`request.form`)              |
| `/phone`         | GET, POST  | Валидация и форматирование телефона           |

## ✅ Логика Cookie

- **Первый визит** → кука `devlab_visited=yes` **устанавливается**
- **Повторный визит** → кука **удаляется**
- Обновите страницу (F5), чтобы увидеть переключение

## 📞 Валидация телефона

Допустимые форматы ввода:
- `+7 (123) 456-75-90` → `8-123-456-75-90`
- `8(123)4567590` → `8-123-456-75-90`
- `123.456.75.90` → `8-123-456-75-90`

Правила:
- Допустимые символы: цифры, `+`, пробел, `(`, `)`, `-`, `.`
- Если начинается с `+7` или `8` → 11 цифр, иначе 10
- Ошибки выводятся через Bootstrap-классы `is-invalid` / `invalid-feedback`
